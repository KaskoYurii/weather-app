# This is a weather APP 
This app created to practice the TS and some Tailwind 
## What used: 
- Vue 3 
- Pinia 
- TS 
- Tailwind 
- https://open-meteo.com/ as data reference
## To run project:
install Docker
- run docker compose up or docker compose up -d
- open http://localhost:5173/ (live server)
